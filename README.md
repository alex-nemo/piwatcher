# PiWatcher tool

This is the source code for the PiWatcher tool that enables you to control your PiWatcher device over I2C.

The PiWatcher is a hardware watchdog for the Raspberry Pi made by [OMZLO](https://www.omzlo.com). 

To learn more go to [the PiWatcher homepage](https://www.omzlo.com/articles/the-piwatcher).
